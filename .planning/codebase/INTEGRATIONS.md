# Integrations

*Last updated: 2026-06-12*

## External APIs
- **Google Gemini API**: Uses `google-genai` for natural language understanding, function calling, and multimodal processing. 

## Local Services
- **Adobe Photoshop**: Integrates via Windows COM (`win32com.client.Dispatch("Photoshop.Application")`). Operates directly on the running Photoshop instance.

## Communication
- **Socket.IO**: Bi-directional real-time communication between the frontend client and the backend agent.
