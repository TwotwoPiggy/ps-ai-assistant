# Tech Stack

*Last updated: 2026-06-12*

## Languages
- Python 3.10+ (Backend)
- TypeScript/HTML/CSS (Frontend)

## Frameworks & Libraries
- **Backend**: FastAPI, uvicorn (Web Server)
- **Frontend**: React, Vite (Build Tool)

## Dependencies
- **Backend**: 
  - `google-genai`: Gemini API integration
  - `pywin32`: Windows COM interface for Photoshop
  - `python-socketio`: WebSocket communication
- **Frontend**:
  - `socket.io-client`: WebSocket client

## Runtime
- Node.js (Frontend development)
- Python (Backend server)
- Windows OS (Required for `pywin32` COM interface)
