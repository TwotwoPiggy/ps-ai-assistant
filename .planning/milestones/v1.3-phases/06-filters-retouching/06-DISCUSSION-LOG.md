## 2. 商业磨皮安全性

| Option | Description | Selected |
|--------|-------------|----------|
| 静默自动新建备份图层或应用智能滤镜 | 无需前置频繁反问打扰用户。如果当前是智能对象则直接应用智能滤镜；如果是普通图层则自动复制一个名为 [原图层名]_磨皮 的图层并在其上磨皮，保证原图无损 | ✓ |
| 强制前置人机交互确认 | 类似于 Phase 05，遇到磨皮口令时大模型必须反问确认：“即将对图层进行磨皮操作，是否使用无损备份图层？”，由用户决策后再行操作 | |
| 直接应用在当前图层上 | 直接在当前图层执行，破坏原始像素，不创建备份 | |

**User's choice:** 静默自动新建备份图层或应用智能滤镜
**Notes:** 无需频繁反问用户，静默复制新图层实现无损操作。

## 3. Camera Raw 与神经元滤镜调用

| Option | Description | Selected |
|--------|-------------|----------|
| 支持外部 .xmp 文件路径参数 + 内置默认预置；神经网络滤镜错误时优雅降级 | 支持外部自定义 .xmp 文件绝对路径，如果没有传入则使用内置的默认常用预置（胶片、日系等）；遇到神经元滤镜不可用或报错时，严格捕获异常并由大模型优雅向用户解释并引导降级到传统滤镜或宏磨皮动作 | ✓ |
| 仅支持预置风格加载 | 不开放自定义 .xmp 路径参数，神经元滤镜失败时直接抛出系统原生异常错误 | |

**User's choice:** 支持外部 .xmp 文件路径参数 + 内置默认预置；神经网络滤镜错误时优雅降级
**Notes:** 保证了专业用户的自定义 ACR 预设工作流，并通过异常捕捉实现 Neural Filters 优雅降级。

## 4. 生成式填充策略

| Option | Description | Selected |
|--------|-------------|----------|
| 无选区时严格拦截报错并引导；提示词自动英译并进行 Prompt 扩充优化 | 底层检测到没有活动选区立即拦截报错并指导用户绘制选区；提示词由大模型自动英译，并根据用户简单意图扩充修饰词以提升生成质量 | ✓ |
| 无选区时自动执行“全选”；提示词进行严格直译 | 无选区时默认执行 Select All，提示词严格直译，不做修饰扩充 | |
| 由 AI 决定 | 由大模型在运行时自由决定选区与提示词处理 | |

**User's choice:** 无选区时严格拦截报错并引导；提示词自动英译并进行 Prompt 扩充优化
**Notes:** 避免全画布生成造成的不可控破坏；通过提示词优化让生成的图片质量更好、更符合语境。

## 6. 磨皮分辨率自适应

| Option | Description | Selected |
|--------|-------------|----------|
| 后端根据 DPI/画布尺寸自动进行参数自适应换算 + 允许显式参数覆盖 | 后端根据当前文档分辨率与默认标尺自动等比缩放高反差和表面模糊半径，确保在不同大小的图像上呈现相同级别的质感，同时大模型可传入自定义参数覆盖 | ✓ |
| 完全交由大模型实时计算 | 后端不做任何换算，大模型每次需先获取尺寸，在大脑中推理计算后下发 | |
| 采用完全固定的静态值 | 后端写死高反差 4px、表面模糊 8px 等静态常数，不考虑画幅大小 | |

**User's choice:** 后端根据 DPI/画布尺寸自动进行参数自适应换算 + 允许显式参数覆盖
**Notes:** 采用自适应的滤波像素值，防止小图浆糊或大图失效。

## 9. Camera Raw 安全保护

| Option | Description | Selected |
|--------|-------------|----------|
| 自动转换为“智能对象”后再应用 Camera Raw 滤镜 | 保证滤镜作为“智能滤镜”挂载，用户在 PS 侧可以双击无限次重调，实现真正的专业级无损修图流程 | ✓ |

**Notes:** 采用行业标准的 Smart Object 流程应用 Camera Raw。

## 10. 滤镜选区边界

| Option | Description | Selected |
|--------|-------------|----------|
| 默认尊重选区 + 提供可选的 clear_selection 参数 | API 默认直接应用，效果受选区局限。但接收可选参数 clear_selection，当大模型判断用户意图为全局模糊/锐化时，可通过传递此参数在应用滤镜前清空选区，减少往返 | ✓ |

| 由 AI 决定 | 大模型自行选择细节 | |

**User's choice:** 默认尊重选区 + 提供可选的 clear_selection 参数
**Notes:** 在尊重 Photoshop 原生选区规则前提下，添加可选的一键清空防误触逻辑。
