<objective>
Implement lifecycle management scripts (install, update, uninstall).

Purpose: Enable users to easily deploy, update, and remove the PS AI Assistant without manual command line operations.
Output: `install.bat`, `scripts/install.ps1`, `updater.py`, `uninstall.bat`, and corresponding test stubs.
</objective>

<threat_model>
- **Risk 1:** Malicious overwrite during update via MITM. **Mitigation:** Rely on HTTPS endpoints for Github ZIP downloads and Git operations.
- **Risk 2:** Unintended file deletion during uninstallation. **Mitigation:** The uninstall script explicitly deletes the designated install directory only, avoiding broad recursive sweeps.
- **Risk 3:** Arbitrary command execution if updater.py is invoked with poisoned inputs. **Mitigation:** Updater accepts no external inputs.
</threat_model>
